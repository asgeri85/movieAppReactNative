# Movies-App

### Movies application with React Native

## Project Features
 - React Navigation 
 - Apisauce
 - React Native Config
 - React Native Swipeable List
 - React Native Svg
 - Redux, React Redux
 - Lottie React Native
 - React Native Modal
 - React Native Vector Icons
 - React Native Youtube Iframe
 - React Native Image Slider Show
